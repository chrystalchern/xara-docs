
setNodeDisp
^^^^^^^^^^^


.. py:method:: Model.setNodeDisp(tag, dof, value)

   Sets the displacement of a node in a given degree of freedom.

   :param tag: The tag of the node to set the displacement for.
   :type tag:  |integer|
   :param dof: The degree of freedom to set the displacement for (0 for x, 1 for y, 2 for z).
   :type dof: |integer|
   :param value: The displacement value to set.
   :type value: |float|
