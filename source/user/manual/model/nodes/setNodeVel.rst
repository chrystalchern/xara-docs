
setNodeVel
^^^^^^^^^^

.. py:method:: Model.setNodeVel(tag, dof, value)

   Sets the velocity of a node in a given degree of freedom.

   :param tag: The tag of the node to set the velocity for.
   :type tag:  |integer|
   :param dof: The degree of freedom to set the velocity for (0 for x, 1 for y, 2 for z).
   :type dof: |integer|
   :param value: The velocity value to set.
   :type value: |float|
