
setNodeAccel
^^^^^^^^^^^^^

.. py:method:: Model.setNodeAccel(tag, dof, value)

   Sets the acceleration of a node in a given degree of freedom.

   :param tag: The tag of the node to set the acceleration for.
   :type tag:  |integer|
   :param dof: The degree of freedom to set the acceleration for (0 for x, 1 for y, 2 for z).
   :type dof: |integer|
   :param value: The acceleration value to set.
   :type value: |float|
